# LID-DS

Intrusion detection system for analyzing CVE-2014-0160 (Heartbleed) attacks using the LID-DS


**Scenario Path**: 

Insert the path for the chosen scenario, for example: `SCENARIOS/CVE-2014-0160`

### Setup 

```bash
pip install -r requirements.txt
```

```bash
python3 setup.py
```

The setup:
- Clone the LID-DS repository 
- Run an initial test to verify everything 

### Run Detection

After setup is complete, it runs the main script

If you want to run the detection script manually:

```bash
python3 testscen.py
```
